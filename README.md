# Back End Template
# Built with Node, Express, and MongoDB, configured to accept JSONs with corresponding name and timestamp.
# 1. Type 'git clone https://github.com/andr00js/backendTemplate' in terminal to clone this repo into the current folder open in terminal.
# 2. Download node.js
# 3. Run 'npm install' in 'backendTemplate' folder to load repo dependencies.
# 4. Create file name '.env' in 'backendTemplate'. Save MongoDB url in a variable named 'DATABASE_URL'. Save TCP port number in variable named 'PORT'.
# 5. Run 'npm run server' in 'backendTemplate' folder to create a server instance.
# 6. Open 'localhost:PORT' in browser, using the TCP port number selected in step 4 in place of PORT.
